# EVASION (VILDMARK) — Landing Page E-Commerce Premium untuk Peralatan Outdoor Pintar

**EVASION** (atau dikenal juga sebagai **VILDMARK**) adalah sebuah template landing page e-commerce premium, interaktif, dan modern yang dirancang khusus untuk merek perlengkapan luar ruangan kelas atas (high-end outdoor gear) dan aksesoris pintar (smart accessories). 

Halaman web ini menampilkan desain estetis bergaya minimalis nan mewah dengan berbagai macam interaksi berbasis gerakan scroll (*scroll-driven animations*) yang halus dan memukau, memberikan pengalaman pengguna (*user experience*) yang sangat imersif bagi calon pelanggan.

## 🌟 Fitur Utama
Aplikasi ini dikembangkan dengan fokus pada visual yang menawan dan performa animasi yang sangat responsif:

1. **Parallax Bento Grid Hero Section**
   Efek transisi di bagian Hero di mana tata letak gambar (bento grid) merenggang dan mengecil secara dinamis berdasarkan posisi scroll pengguna.
2. **Interactive Product Split (Alpine & Forest)**
   Transisi split-screen yang mempertemukan dua produk andalan (Backpack "Alpine" & Thermal Mug "Forest") dari sisi kiri dan kanan layar seiring pengguna melakukan scroll ke bawah.
3. **Smooth Scroll-Reveal Text**
   Animasi pewarnaan teks paragraf deskripsi teknologi kata demi kata secara halus saat masuk ke area pandang (*viewport*).
4. **Horizontal Image Gallery**
   Galeri produk termal horizontal imersif yang bergerak ke kiri dan kanan berdasarkan kontrol scroll vertikal halaman.
5. **Modern Essential Accessories Section**
   Grid dan carousel responsif yang menyajikan daftar aksesori petualangan pintar premium lengkap dengan detail harga dan deskripsi singkat.
6. **Desain Responsif & Estetis**
   Menggunakan palet warna gelap/terang yang harmonis, sudut membulat elegan (*smooth border radius*), efek blur kaca (*glassmorphism*), dan tipografi modern dengan font **Inter**.

## 💻 Tech Stack (Teknologi yang Digunakan)

* **Framework**: [Next.js 16.0.10](https://nextjs.org/) (App Router)
* **Programming Language**: [TypeScript](https://www.typescriptlang.org/)
* **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) dengan `@tailwindcss/postcss` & `tw-animate-css`
* **UI Components**: [Radix UI Primitives](https://www.radix-ui.com/) (Accordion, Dialog, Tabs, Dropdown Menu, dsb.)
* **Icons**: [Lucide React](https://lucide.dev/)
* **Analytics**: [@vercel/analytics](https://vercel.com/analytics)

## 🚀 Cara Menjalankan Proyek Secara Lokal

Pastikan Anda telah menginstal **Node.js** di komputer Anda.

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/alifconstantine/Evasion-eCommerce.git
   cd evasion-e-commerce-template
   ```

2. **Instal dependensi proyek:**
   Anda dapat menggunakan `npm` atau `pnpm`:
   ```bash
   npm install
   # atau
   pnpm install
   ```

3. **Jalankan server pengembangan (Development Server):**
   ```bash
   npm run dev
   # atau
   pnpm dev
   ```
   Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya secara langsung.

4. **Build untuk produksi:**
   ```bash
   npm run build
   npm run start
   ```
